# vimrc

## 1. Set up [Vundle]:

  ` git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`

## 2. Set up [User .vimrc]:

  ` git clone https://github.com/paranlee/vimrc.git ~/vimrc`

  ` cp ~/vimrc/.vimrc ~`

## 3. Install Plugins:

  Launch `vim` and run `:PluginInstall`
